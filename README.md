# StorageBox Adapter
This mod is an addon that adds adapters for StorageBox.<br>
It allows items to be inserted and extracted from StorageBox using hoppers and industrial mod devices.

This is an addon for [StorageBox Mod](https://www.curseforge.com/minecraft/mc-mods/storagebox-mod) that works with Forge/NeoForge, and also supports the Fabric version [StorageBox for Fabric](https://www.curseforge.com/minecraft/mc-mods/storagebox-fabric).
